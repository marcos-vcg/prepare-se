# PREPARE-SE
Projeto criado para preparadores sobrevivencialistas.

## 🖥️ Sobre o projeto

O aplicativo serve como um guia para sobrevivencialistas poderem adiquirir maiores informações e se organizar melhor. 


## 🖼️ Layout

- [ ] _WEB_

- [x] _MOBILE_


## 🌎 BUILD & RUN
* Instalar as Dependências: 

      npm install

* Rodar o Projeto: 

      npx expo start



## 🛠 Tecnologias

* React Native, 
* Typescript, 
* TailwindCSS
